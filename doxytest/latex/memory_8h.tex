\subsection{Detailed Description}
Provides the memory related Phasor scripting functions.

Note\-: As of 01.\-00.\-10.\-059 all read/write functions have been overloaded to optionally remove the need for the address offset. For such cases you shouldn't include address\-\_\-offset and the address used will be exactly what's specified and not (base\-\_\-address + address\-\_\-offset) as in previous versions.

For example, the below are equivalent 
\begin{DoxyCode}
readbyte(12345, 5) 
readbyte(12350)
\end{DoxyCode}


\paragraph*{D\-E\-F\-I\-N\-I\-T\-I\-O\-N\-S}

byte -\/ 8 bit positive integer from \mbox{[}0,255\mbox{]}

char -\/ 8 bit integer from \mbox{[}-\/128, 127\mbox{]}

word -\/ 16 bit positive integer from \mbox{[}0,65,535\mbox{]}

short -\/ 16 bit integer from \mbox{[}-\/32768, 32767\mbox{]}

dword -\/ 32 bit positive integer from \mbox{[}0, 4,294,967,295\mbox{]}

int -\/ 32 bit integer from \mbox{[}-\/2147483648, 2147483647\mbox{]}

float -\/ 32 bit floating point number from \mbox{[}-\/3.\-402823466e+38, 3.\-402823466e+38\mbox{]}
